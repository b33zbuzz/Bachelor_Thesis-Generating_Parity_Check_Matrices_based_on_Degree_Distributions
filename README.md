# Bachelor_Thesis-Generating_Parity_Check_Matrices_based_on_Degree_Distributions
Optimizing LDPC codes via node degree distributions. Features a Python/NumPy framework for synthesizing large parity-check matrices based on IRA code research. Includes h5py-based matrix visualization and validation against fractional edge models to ensure reliable data transfer in modern digital communication systems.

Parity check (H) matrices for LDPC codes are randomly generated and sometimes have inefficiencies. Is there a way to generate a more efficient H-matrix? In the modern age, digital communication requires increasingly stable connections and accurate, reliable data transfers. As the frequency and complexity of these transfers grow, the development of advanced error detection and correction algorithms becomes essential. Low-Density Parity-Check (LDPC) codes are a prevalent type of error-correcting code used to enhance data reliability. This thesis explores potential improvements to basic LDPC codes by examining the degrees of check nodes and variable nodes. Drawing inspiration from the paper "Irregular Repeat–Accumulate Codes," which posits that a specific fractional distribution of edges provides robust encoding, this study evaluates the effectiveness of these distributions. The results aim to contribute to the enhancement of LDPC codes, thereby supporting more reliable digital communication systems.
